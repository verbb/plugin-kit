# TiptapEditor

TiptapEditor provides a rich-text editing surface with a configurable toolbar, dynamic variable insertion, and structured JSON output.

## Basic Usage

Use a focused toolbar for the default writing experience and keep the editor controlled so value changes stay in sync with the surrounding form.

## Expanded Toolbar

Use a broader toolbar when the editor is the primary authoring surface and needs headings, tables, links, code, and undo or redo support.

Variable categories let the editor expose dynamic placeholders without requiring users to remember token syntax.
