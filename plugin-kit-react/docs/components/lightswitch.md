# Lightswitch

Lightswitches communicate immediate on or off state for binary settings.

## Basic Usage

Use a lightswitch for binary settings where the current state should be immediately visible.

## Disabled

## API reference

`Lightswitch` wraps [Base UI Switch](https://base-ui.com/react/components/switch/). Use the Base UI API reference for checked state, disabled behavior, form integration, and switch accessibility details.
