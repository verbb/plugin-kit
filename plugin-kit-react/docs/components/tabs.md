# Tabs

Tabs organize related content into compact sections without leaving the current view.

Use standard tabs for the most common inline section switching pattern.

## Pane Tabs

`Tabs` wraps [Base UI Tabs](https://base-ui.com/react/components/tabs/). Use the Base UI API reference for tab activation, keyboard behavior, disabled tabs, and controlled state.
