# Copy Button

Copy buttons make one-click copying obvious for URLs, tokens, IDs, and other read-only values. They usually sit alongside an input-like field that shows the copied value.

## Basic Usage

Pair the copy action with a visible read-only field so users can verify the value before copying it.
